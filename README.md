# My page app

```
django=5.2.4
python=3.13.1
```

### Create venv

```
python3 -m venv .venv
source .venv/bin/activate
pip install django
```

### Start project

```
django-admin startproject mypage .
```

### Run in the local server

```
python3 manage.py runserver
```

### Create app

```
python3 manage.py startapp challenges
```
